# Atlas Careers Redesign

GitHub Pages–ready static careers mockup inspired by the Ather Labs careers artifact.

## Files
- `index.html`
- `styles.css`
- `script.js`

## Publish to GitHub Pages
1. Upload this folder to your repo root.
2. Go to **Settings → Pages**.
3. Set **Source** to `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Save.

Your site URL will look like:

```txt
https://duongduongblue.github.io/atlas-careers-redesign/
```
